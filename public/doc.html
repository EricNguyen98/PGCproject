PG County Office Project

Brief Summary

This project makes use of HTML, CSS, JS. A back end is used to process data from the PG County API and pass it to the front end where it is read as a JSON object and the data is loaded onto the page. The backend is written in JS and runs in Node.js. The backend supports 3 API calls currently GET is purposed for returning all PG County offices, PUT is used to return offices that match the agency condition for the map, and POST is used to return offices that match the agency condition for the list. JSON objects are then parsed frontend to produce the list and map seen.
